Vim Recipes

Vim Recipes is a free cookbook for Vim, the popular text editor. It's very much a work in progress, but is available in:

  • PDF format (good for printing or downloading).
  • HTML format (good for reading on-line).
  • Bziped Tarball format (good for downloading on Linux/Mac).
  • Zip format (good for downloading on Windows).
   

You can subscribe to the Atom feed, or follow @vimrecipes on Twitter, to be notified of new recipes.

Please report any problems, typos, and suggestions via the issue tracker or e-mail to runrun@runpaint.org.

The latest source for the book is available from GitHub.
