Searching the Vim Documentation

Problem

You want help with Vim but don't know where to look.

Solution

Use the :helpgrep pattern command.

Discussion

To look up documentation for a specific Vim feature you can use :help feature-name. But this is only useful if you know the name of the feature.

helpgrep takes a pattern and matches it against the locally installed documentation in much the same way as vimgrep did in Searching Over Multiple Files. If it finds any matches, it adds them to the quick fix listSee the Quick Fix List sidebar in Searching Over Multiple Files for more information, and jumps to the first one.

Phrases in the help files in a different font face to the rest of the text are hyperlinks to related documentation. To follow these links use <Ctrl>+] when your cursor is over one.

If you've installed a Vim addon, you'll need to run :helptags docs-path before helpgrep will see its documentation.
